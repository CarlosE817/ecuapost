# EcuaPost 🇪🇨

**EcuaPost** es una red social minimalista tipo Twitter, desarrollada con:

- ⚛️ React + TypeScript
- ⚡ Vite
- 🎨 TailwindCSS

---

## ✨ Funciones

- Crear y visualizar publicaciones tipo tweet
- Likes y retweets
- Interfaz responsiva y limpia
- Datos simulados (`mockData.ts`)
- Personalización de usuario

---

## 🚀 Instalación

```bash
npm install
npm run dev
